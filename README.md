# ArtXHacking
당신의 자아를 제어합니다.

자신이 통제하기 힘든 현대인의 또 다른 자아 "스마트폰"을 

스스로 타인의 통제 하에 두려는, 해킹과 미디어 아트가 결합된 시도입니다.

일반 앱은 다른 기기 뿐만 아니라 자신 조차도 전원 제어가 불가능하지만, "해킹"을 통해 가능합니다.
